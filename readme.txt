== Description ==
You don't have cpanel access?Need to check database table stucture, field value or run custom sql or need to export database?Then simply use this pluing.By this plugin you can see database table structure, database table fields with values and also you can run custom sql from admin panel of your wordpress site and can export database.

== Installation ==

**Normal installation**

1. Download the plugin zip file to your computer.
1. Unzip the file.
1. Upload the unzipped file to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
